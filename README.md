# Supplementary Material for “SkeDiff: Skeleton 3D CT Diffusion Reconstruction using 2D X-ray”

This repository hosts the supplementary material for the paper:

**SkeDiff: Skeleton 3D CT Diffusion Reconstruction using 2D X-ray**

The material provided here contains content moved from the main manuscript during the proof revision process in order to comply with the **14-page limit for a Regular Paper** in **IEEE Journal of Biomedical and Health Informatics (JBHI)**.  
These materials are supplementary in nature and do **not** affect the core methodology, main experimental results, or the principal conclusions of the paper.

## Contents

The supplementary file includes the following sections:

- **A. Computational Resources and Efficiency Tradeoffs**
- **B. Robustness to Image Quality Degradation**
- **C. Training Stability and Practical Considerations**

It also contains the associated supplementary tables and figure, including:

- hyperparameter sensitivity analysis for batch size
- variance across random seeds
- training/validation convergence curves

## Files

- `SkeDiff_JBHI_Supplementary.pdf` — supplementary document containing the additional discussion, tables, and figure moved from the main manuscript.
